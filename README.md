## âš™ï¸ Installation & Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/nerdy-enough.git
    cd nerdy-enough
    ```

2.  **Install Dependencies**:
    ```bash
    # Install backend dependencies
    cd backend
    npm install

    # Install frontend dependencies
    cd ../frontend
    npm install
    ```

3.  **Environment Variables**:
    Create a `.env` file in `backend` and `frontend` directories with your configuration (Mongo URI, JWT Secret, Payment Keys, etc.).

4.  **Run the Application**:
    ```bash
    # Run backend
    cd backend
    npm start

    # Run frontend
    cd frontend
    npm run dev
    ```
