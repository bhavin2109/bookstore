# Contributing to Nerdy Enough

Thank you for your interest in contributing to **Nerdy Enough**! We welcome contributions from the community to help improve this **MERN Stack Bookstore Project**.

## How to Contribute

1.  **Fork the Repository**: Click the "Fork" button at the top right of this page.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/yourusername/nerdy-enough.git
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make Changes**: Implement your feature or fix a bug.
5.  **Commit Changes**:
    ```bash
    git commit -m "feat: Add new feature"
    ```
6.  **Push to Branch**:
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Open a Pull Request**: Submit a PR to the `main` branch of the original repository.

## Guidelines

- Follow the existing code style.
- Write clear commit messages.
- Ensure your changes do not break existing functionality.
- Update documentation if necessary.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the repository.
